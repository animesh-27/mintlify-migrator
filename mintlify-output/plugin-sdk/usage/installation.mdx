Dyte's Plugin SDK allows you to create your own real-time applications that work
seamlessly with Dyte meetings. It comes with many interesting APIs and features
to help you solve complex problems in minuites.

You can use the install the package using NPM, Yarn or CDN.



<Tabs>


  

<Tab title="Tab">



```shell
npm install @dytesdk/plugin-sdk
```

  

</Tab>


  

<Tab title="Tab">



```shell
yarn add @dytesdk/plugin-sdk
```

  

</Tab>


  

<Tab title="Tab">



```html
<script src="https://cdn.jsdelivr.net/npm/@dytesdk/plugin-sdk" />
```

  

</Tab>
