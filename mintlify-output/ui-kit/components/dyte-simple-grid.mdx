---
title: "Dyte Simple Grid"
---

A grid component which renders only the participants in a simple grid.

```html
<dyte-simple-grid id="dyte-el"></dyte-simple-grid>

<script>
  const el = document.getElementById('dyte-el');
  el.participants = [meeting.self];
</script>

<style>
  dyte-simple-grid {
    height: 360px;
    width: 100%;
  }
</style>
```
