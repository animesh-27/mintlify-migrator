---
title: "Dyte Recording Indicator"
---

A component which indicates the recording status of a meeting.

It will not render anything if no recording is taking place.

```html
<dyte-recording-indicator id="dyte-el"></dyte-recording-indicator>

<script>
  const el = document.getElementBydId('dyte-el');
  el.meeting = meeting;
</script>
```
