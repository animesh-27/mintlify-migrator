title: "Dyte Audio Visualizer"
---

An audio visualizer component which visualizes a participants audio.

Commonly used inside `dyte-name-tag`.

```html
<dyte-audio-visualizer id="el"></dyte-audio-visualizer>
<script>
  const el = document.getElementById('el');
  el.participant = meeting.self;
</script>
```
